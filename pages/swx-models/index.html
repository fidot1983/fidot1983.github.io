Thanks to Bill Rose from the forum, I have collected a good set of standard hardware models for various bolts, nuts, bearings, rod ends, and such.

Ive added a few models of mine.

Please let me know (contact me on Biplane Forum) (my nickname is 'fidot') if you find any problems or inaccuracies.

All models are available as a single zip file here.

All models are provided in GitHub GIT repository. That allows me to track changes and update files. Think of is as a fancy network file share.

GIT repository can be browsed here.
